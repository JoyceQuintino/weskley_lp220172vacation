LISTA CONTENDO 10 COMANDOS DO GIT

$ git init <nome_projeto> (cria reopistório local)
$ git clone <url> (clona repositorio que está em um servidor para a máquina local)
$ git status (exibe os novos arquivos ou arquivos modificados de um repositório)
$ git add <arquivo> (adcionar arquivo(s) para ser(em) versionados)
$ git commit -m "<mensagem>" (manda os aqruivos adcionados para a staged area, acompanhada de uma mensagem descritiva)
$ git diff (exibe o que foi alterado  no arquivo antes de enviá-lo para a staged area)
$ git log (exibe todos os commits já feitos)
$ gitk (abre o visualizador de relatórios de alterações do git)
$ git checkout -b <nome_branch> (cria um novo branch e seleciona-o)
$ git merge <arquivo> (mescla as alteraçoes entre branchs)
